# WS_demo

## Dependencies
- Tested Cuda versions: 9.0, 10.0

### Python packages

#### For the tracker
- numpy==1.13.1
- opencv-python==3.3.0.10
- tensorflow-gpu (tested versions: 1.5.0 for Cuda 9.0 and 1.13.1 for Cuda 10.0) 

## Installation
```
git clone https://github.com/Arya07/WS_demo.git
cd WS_demo
mkdir build
ccmake ../
make 
make install
```
